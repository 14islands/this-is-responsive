Strategy

Getting Started

Understand the business opportunities of responsive design.

Landscape

Approach

Understand your options

Recognize that responsive design is a subset of adaptive design

Think Future-Friendly

Design

Layout

Design Systems, not Pages

Start Mobile-First: Address contsraints first

Media Queries

Let the content, not device widths, determine breakpoints

Don't assume media query support by default

Build up media queries: more min-widths than max-widths

Don't arbitrarily add breakpoints

Be careful of manipulating the DOM for presentation purposes

Understand matchMedia

Typography

Ensure that type is readable at different viewing lengths

-webkit-text-size-adjust: 100%;

Images

Apply max-width: 100% to images, videos and options to create flexible media

Avoid putting text in images

Ensure content is legible on small screens

Be mindful of the number of images on a page

Adaptive Images

Load the small version of the image by default

Only make one request per image

Testing

Understand the difference between support and optimization

Establish a support matrix

Performance

Follow basic performance best practices

Avoid sending down resources that will never be used by small screen devices

Look for opportunities to conditionally load content
